Big O Notation

Big O notation is used to describe the efficiency of an algorithm as the size of its input grows.

Now there are two metrics which big O notation measures, time complexity and space complexity:

  • The time complexity is defined by the number of operations performed by an algorithm relative to the size of the input.

  • The space complexity is defined by the amount of memory required by an algorithm relative to the size of the input.

The function OOO takes an argument representing the growth rate of the time or space complexity relative to the input size nnn.
